<h1>Boolzap</h1>
<p>Replica web di Whatsapp.</p>

<h2>Tecnologie usate</h2>
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JAVASCRIPT</li>
<li>JQUERY</li>
</ul>

<h2>Caratteristiche</h2>
<ul>
<li>Multi-Chat</li>
<li>Campo di ricerca</li>
<li>Eliminazione di un messaggio con apertura popup</li>
<li>Chat-Bot</li>
<li>Night-Mode</li>
</ul>
